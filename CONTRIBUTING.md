# Contributing

Thank you for your interest in contributing.

Before a pull request can be merged, you must sign the Rockbound Capital
Individual Contributor License Agreement ([CLA.md](CLA.md)). The CLA grants
Rockbound Capital the right to license your contribution under other terms,
including commercial licenses, which keeps dual licensing of this project
possible.

Automatic pull-request gating with
[CLA assistant](https://github.com/cla-assistant/cla-assistant) is planned but
not yet enabled. Once enabled, CLA assistant will comment on your first pull
request with a link to sign electronically, and the check must pass before
merge. Until then, maintainers verify CLA status manually before merging:
email monte@rockbound-capital.com to sign.

By submitting a contribution you confirm that you have the right to license
the contributed code and that it contains no third-party code under
incompatible terms.
